# 使用 Echo#File()

`Echo#File(path, file string)` 使用路径注册新路由以提供静态文件。

用法 1

使用 `public/index.html` 提供索引页面

```go
e.File("/", "public/index.html")
```


用法 2

使用 `images/favicon.ico` 提供一个图标

```go
e.File("/favicon.ico", "images/favicon.ico")

```
